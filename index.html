A-Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

O ::placeholderseletor seleciona elementos de formulário com texto de espaço reservado e permite estilizar o texto de espaço reservado, alterar a cor do texto.


B-Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

type="radio" = Exibe um botão de opção (para selecionar uma das muitas opções) type="checkbox"= Exibe uma caixa de seleção (para selecionar zero ou mais opções)


C- Na tag <input> para que serve o atributo "value"?

É usado para definir o valor inicial ou o valor atual de um campo de entrada em um formulário HTML. Ele preenche o campo de entrada com um valor específico quando a página é carregada.


D- Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

O atributo "name" deve ser usado para agrupar botões de rádio em um formulário, permitindo que apenas um deles seja selecionado de cada vez. Ao atribuir o mesmo valor para o atributo "name" em cada botão de rádio do grupo, somente uma opção poderá ser marcada, desmarcando automaticamente as outras opções.


E- Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

A tag <label> é usada para criar rótulos associados a elementos de entrada em um formulário HTML. O atributo "for" nessa tag é usado para estabelecer a associação entre o rótulo e o elemento de entrada correspondente. Ao clicar no rótulo, o elemento de entrada associado é ativado, facilitando a interação com o formulário.


F- Na tag <option> para que serve o atributo "selected"?

O atributo "selected" na tag <option> é usado para indicar qual opção deve ser selecionada por padrão em um elemento <select> em um formulário HTML. Quando a página é carregada, a opção com o atributo "selected" é exibida como selecionada.


G- Nas tags de formulário para que serve o atributo "required"?

O atributo "required" é utilizado nas tags de formulário HTML para indicar que um campo de entrada deve ser preenchido antes do envio do formulário. Ele garante que o campo seja obrigatório, impedindo o envio se estiver vazio. Isso ajuda a validar e assegurar a integridade dos dados inseridos pelos usuários nos formulários.


H- Para que serve a tag <textarea>? A tag <textarea>em HTML é utilizada para criar uma área de texto multi-linhas em um formulário ou página da web. Ela permite que os usuários insiram um texto mais longo, como comentários, descrições ou mensagens extensas.


I- Na tag <form>para que serve o atributo "action"?

É utilizado para especificar o destino para onde os dados do formulário serão enviados quando o formulário for submetido. Ele define o URL ou o servidor que receberá e processará os dados do formulário. Ao submeter o formulário, os dados são enviados para o URL especificado no atributo "action". Isso permite que o servidor realize ações apropriadas, como armazenar os dados em um banco de dados, enviar e-mails ou processar a lógica de negócio relacionada ao formulário.


J- Na tag <form> para que serve o atributo "method"?

É usado para especificar o método de envio dos dados do formulário quando ele é submetido.Existem dois valores principais para o atributo "method":

GET: Os dados do formulário são anexados à URL como parâmetros de consulta. É visível na barra de endereço do navegador e é adequado para formulários não sensíveis ou para fins de pesquisa.

POST: Os dados do formulário são enviados no corpo da requisição HTTP, não sendo visíveis na URL. É mais adequado para formulários com dados sensíveis ou quando o envio de dados pode modificar o estado do servidor.


K- Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

O atributo "method" na tag <form> deve ser definido como "POST".


L- Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

Deve ser definido como "POST".O método "POST" é o mais apropriado para a transmissão segura dos dados do arquivo para o servidor. Além disso, é necessário incluir o atributo "enctype" com o valor "multipart/form-data".


M- O que as tags <fieldset> e <legend> produzem?

A tag <fieldset> cria um contêiner visual que envolve os campos relacionados, permitindo agrupá-los logicamente.

A tag <legend> é inserida dentro da tag <fieldset> e fornece uma legenda ou título descritivo para o grupo de controles.
